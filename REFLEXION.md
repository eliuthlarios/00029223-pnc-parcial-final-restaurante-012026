Primero revisé el README para entender qué se pedía. Luego le pedi a char ir organizando el proyecto en entidades, repositorios, servicios, controladores y DTOs

Una de las partes más importantes fue la seguridad. Se manejaron roles como administrador, encargado de turno y cliente. También se usó JWT y refresh token para el inicio de sesión y el acceso a endpoints protegidos.

También se agregó una regla de negocio para que el proyecto no fuera solamente CRUD.El encargado de turno solo puede administrar pedidos y mesas de la sucursal a la que pertenece.
